# TorAPI Instance

Public instance of [Ryuk-me/Torrent-Api-py](https://github.com/Ryuk-me/Torrent-Api-py)
for the Torrent Music Player app.

## Deploy to Render (free)

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/Komsomol39/torapi-instance)

## API

After deploy, use your URL in Settings → RU Torrents → Instance URL

Example: `https://your-app.onrender.com`

Test: `GET https://your-app.onrender.com/api/v1/search?site=1337x&query=test&limit=3`
